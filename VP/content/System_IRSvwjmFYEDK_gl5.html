Name Description
 Agendar cita : Use Case

El sistema permitirá al cliente o administrador crear una nueva cita seleccionando fecha, hora, servicio y profesional disponible

 Validar disponibilidad : Use Case

El sistema verifica franjas de horarios disponibles para el agendamiento de citas de los empleados

 Consultar agenda de citas : Use Case

 El sistema permitirá visualizar la agenda de citas mediante un calendario o lista filtrada por día, profesional o estado de la cita

 UseCase2 : Use Case
 Registrar nuevos servicios : Use Case

El sistema permitirá al administrador registrar nuevos servicios (ej. Manicura, Masaje) especificando nombre, descripción, duración y precio

 Registro de Usuarios : Use Case

El sistema debe permitir el registro de usuarios con sus respectivos roles (como clientes y empleados), mediante sus credenciales (numero de documento y contraseña) y el sistema guarda la información en la base de datos

 Actualizar información de perfil : Use Case

El sistema permitirá actualizar la información del perfil del usuario (teléfono, contraseña, etc)

 Inhabilitar cuenta de usuario : Use Case

 El sistema permitirá realizar la desactivación lógica de una cuenta de usuario cuando este lo solicite o por decisión administrativa

 Buscar perfil de usuario : Use Case

El sistema permitirá buscar y visualizar el perfil de un usuario mediante filtros como nombre, identificación o correo electrónico

 Reprogramar cita : Use Case

El sistema permitirá reprogramar una cita (cambio de horario o especialista) notificando automáticamente a las partes involucradas

 Cancelar cita programada : Use Case

El sistema permitirá cancelar una cita programada, liberando el espacio en el calendario de disponibilidad

 Buscar servicios : Use Case

El sistema permitirá a los clientes navegar y buscar servicios por categorías o palabras clave

 Actualizar detalles de servicios : Use Case

 El sistema permitirá actualizar los detalles de un servicio, como ajustes en el precio o cambios en la duración del procedimiento

 Eliminar servicios del catálogo : Use Case

El sistema permitirá retirar servicios del catálogo o marcarlos como "no disponibles" para que no aparezcan en el agendamiento.

 Generar nuevos reportes de pagos : Use Case

El sistema permitirá generar nuevos reportes personalizados seleccionando el periodo (día/semana/mes) y el criterio (profesional/servicio)

 Visualizar reportes generados de pagos : Use Case

El sistema permitirá visualizar los reportes generados en pantalla con gráfica de los ingresos en el período seleccionado y tabla de datos

 Editar parámetros de reportes generados de pagos : Use Case

El sistema permitirá editar los parámetros de un reporte ya configurado (ej. cambiar el rango de fechas antes de exportar)

 Borrar historial de reportes de pagos generados : Use Case

El sistema permitirá borrar reportes guardados o históricos del panel administrativo para limpieza de datos

 Registrar nuevo pago : Use Case

El administrador registra el medio de pago que utilizan los clientes

 Consultar historial de transacciones : Use Case

 El sistema permitirá consultar el historial de transacciones y el estado de los pagos realizados por cada cliente.

 Anular registro de pago : Use Case

El sistema permitirá anular un registro de pago en caso de errores graves o devoluciones, dejando trazabilidad del motivo.

 Inicio de sesión : Use Case

Para iniciar sesion cada usuario debe ingresar numero de documento y contraseña

 Registrar calificación de servicio : Use Case

 El sistema permitirá al cliente registrar una calificación (estrellas) sobre su experiencia tras finalizar su cita

 Visualizar promedio de calificaciones : Use Case

El sistema permitirá a los usuarios visualizar el promedio de calificaciones de los profesionales, los servicios y el establecimiento

 Editar registro de pago : Use Case

El sistema permitirá corregir errores en el registro de pago (ej. si se marcó efectivo pero fue transferencia) antes del cierre de caja

 Validar datos : Use Case

Validar que las credenciales ingresadas por parte del usuario sean correctas

 Calendario profesional : Use Case

Permmite configurar la disponibiliadad y horarios de trabajo del profesional

 Asignar servicios : Use Case

El sistema permitirá seleccionar varios servicios para asociarlos a una cita

 Crear orden : Use Case

El sistema genera una orden asociada a una cita agendada

 Modificar orden : Use Case

El sistema permitirá al administrador actualizar información antes de cerrar o pagar
